
SR TOP UP Web Order System - Setup Guide

1. Upload all files (index.html, style.css, script.js) to your GitHub repository.
2. Deploy formSubmit.gs script on Google Apps Script:
   - Go to https://script.google.com
   - Paste formSubmit.gs code
   - Save and Deploy as Web App (select "Anyone" access)
3. Replace the URL in script.js with your deployed script URL.
4. Publish your GitHub page and connect it to Telegram Mini App.
